## 1.1 SQL也是语言
Structured Query Language 结构化查询语言,用于访问和处理数据库的标准的计算机语言  
SQL不是编程语言，但是也有自己的标准，有SQL92与SQL99  
## 1.2 SQL的特点
综合统一，不同的数据库可以有自己的语法，但是都支持基本语法  
面向对象（？）总之不用关心内部实现

## 3. 数据库系统
### 数据库
数据库 *database*是一个以某种组织方式存储在磁盘上的数据集合  
### 数据库的应用
各种管理系统电商什么的  
### 数据库软件

#### 关系型数据库*RDBMS*  
Oracle数据库（大型，甲骨文）  
MySQL数据库（最流行的中型数据库）  
SQL server数据库（Windows上比较好的数据库）  
SQLite（嵌入式用的小型数据库）  
#### 非关系型数据库*NoSQL*
Redis（缓存数据库）  
MongoDB(文档型数据库)  
等等(好多，用到再了解吧)

### SQL与数据库软件的关系
SQL是用于操作数据库的语言，适用于全部的RDBMS  
数据库软件一般也会自己扩展专属自己的SQL语法  
但是NoSQL不太适用
